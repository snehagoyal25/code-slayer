# Valid Parentheses

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

### Input
- A string s.

### Output
- Return true if the string is valid, false otherwise.

### Example

**Input:**
s = "()[]{}"

**Output:**
true

### Constraints
- 1 <= s.length <= 104
- s consists of parentheses only '()[]{}'.
